Aide HELPDESK

Comment signaler un problème technique ?

1) Sélectionnez le niveau d'urgence de votre problème (par défaut : l'urgence sélectionnée est : "Dans la journée").

2) Indiquez votre numéro de machine.

Si vous ne connaissez pas votre numéro de machine vous devez cliquer sur l'icone "?" :

  • Un formulaire s'ouvrira et vous devrez indiquer les premières lettres de votre nom, du nom de votre ordinateur ou de son numéro de série avant de cliquer sur "Rechercher".
  • Ensuite une liste devrait apparaitre, vous trouverez votre numéro de machine en face de votre nom. Il vous suffit de cliquer sur la ligne concernée pour mettre à  jour le formulaire.
  • Si votre nom n'apparait pas contactez un responsable informatique.

3) Décrivez votre problème

Soyez le plus précis possible dans votre description. Le traitement n'en sera que plus rapide.

4) Suivi par mail

Il est aussi possible que deux autres champs fassent partie du formulaire :

  • Le champ «  informez moi des suites données   » si vous repondez oui dans ce champ, vous recevrez une notification par e-mail, à  chaque fois qu'un technicien ajoutera un message de suivi à  votre intervention.
  • Le champ «  mon adresse de messagerie   » qui vous permettra de spécifier une adresse de messagerie pour recevoir ces suivis.

Une fois ce formulaire duement rempli cliquez sur «  envoyer message   ».

Vous pouvez également ajouter une nouvelle demande d'intervention en cliquant sur l'icone "Ajouter une nouvelle intervention" en haut à  droite de votre écran.

Comment suivre les demandes d'interventions déjà  envoyées ?

Il vous est aussi possible de suivre en ligne l'évolution des interventions, vous concernant, qui sont en cours. Pour cela cliquez sur l'icone "suivi des interventions" en haut à  droite de votre écran.

Vous devriez voir apparaitre la liste de toutes vos interventions en cours. Pour obtenir plus d'informations concernant chaque intervention, vous pouvez cliquer sur le lien «  infos   »

Comment modifier vos paramètres personnalisés de connection à  l'application ?

La langue dans laquelle vous souhaitez afficher l'interface (par défaut en anglais), si vous voulez une autre langue sélectionnez « votre langue   » puis cliquez sur «  change   ».

Pour modifiez votre mot de passe, il vous suffit de saisir votre mot de passe dans le formulaire correspondant en haut de votre écran.

Comment se déconnecter ?

Pour vous déconnecter de l'application GLPI, il vous suffit de cliquer sur l'icone "Déconnexion" en haut à  droite de votre écran.
